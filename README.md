<h3>Flags </h3>
Flags são registradores onde cada bit significa um estado do processador. São eles:<br/>

CF = Carry <br/>
PF = Parity <br/>
AF = Auxiliar Carry <br/>
ZF = Zero <br/>
SF = Signal <br/>
TF = Trap <br/>
IF = Interrupt Enable Flag <br/>
DF = Direction <br/>
OF = OverFlow <br/>
