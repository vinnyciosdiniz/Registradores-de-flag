<h3>Flags </h3>
Flags são registradores onde cada bit significa um estado do processador.<br/>
São eles:<br/>
C = Carry <br/>
P = Parity <br/>
A = Auxiliar Carry <br/>
Z = Zero <br/>
S = Signal <br/>
T = Trap <br/>
I = Interrupt Enable Flag <br/>
D = Direction <br/>
O = OverFlow <br/>
